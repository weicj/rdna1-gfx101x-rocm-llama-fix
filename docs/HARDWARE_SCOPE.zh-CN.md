# 硬件适用范围

这份项目在概念上面向的是：

- `RDNA1`
- `Navi14`
- `gfx101x`

但这不等于下面所有型号都在本仓库里被逐卡验证过。

更准确的说法是：

- 这些卡属于同一代架构和驱动家族
- 因此本仓库的方法、坑点、修补逻辑对它们更可能有参考意义

## 真实实测环境

这份仓库里的真实验证，主要来自下面这套环境：

- 主机平台：`Mac Pro 5,1`
- 启动栈：`OpenCore`
- 系统：`Ubuntu`
- 显卡：`Radeon Pro W5500`
- ASIC：`Navi14`
- 架构：`gfx1012`

所以仓库中的性能数字、部署结论、限制条件，都是 anchored 在这条真实 `W5500` 实测线上，而不是凭空外推。

## 风险提示

这份仓库更适合被理解成：

- 一份高价值的实战经验
- 一份真实工程笔记
- 一套在同架构家族里很可能有帮助的方法论

但它不应该被理解成：

- 一张官方支持矩阵
- 对所有板卡和桥接拓扑的成功承诺
- 对任意 BIOS、固件、内核与 PCIe 链路状态的通用保证

如果你要在其它 `RDNA1 / Navi14 / gfx101x` 卡上复用这套方法，请把它当作强参考，而不是无条件保证。风险自担。
